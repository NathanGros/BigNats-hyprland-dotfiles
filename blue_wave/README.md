# Blue Wave

## Dependencies
Make sure to have the following packages and utilities installed:
- btop
- cava
- checkupdates-with-aur
- fastfetch
- foot
- hypridle
- hyprland (obviously)
- hyprlock
- mako
- neovim
- oh-my-posh
- swww
- udiskie
- waybar
- wl-clipboard
- wlogout
- wofi
- yay
- zsh
